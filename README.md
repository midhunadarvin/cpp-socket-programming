# Socket Programming in CPP

### 1. Socket Chat app

**Start Server**
    
    g++ -o Server.exe Server.cpp 
    ./Server.exe 8000
    
**Start Client**

    g++ -o Client.exe Client.cpp
    ./Client.exe 127.0.0.1 8000
